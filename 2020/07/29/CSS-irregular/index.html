不規則邊緣

HTML

Code

CSS

文字內容區塊要注意的地方是記得加上 position: relative;、z-index: 2;，讓 container 內的內容顯示在最上層(畫面上)。

不規則的泡泡邊框一樣用 :after 來處理，先做出一個圓形區塊，使用 transform: translateX(-30%) translateY(-50%); 移動到邊緣。接下來其他的泡泡部分則用 box-shadow 來產生。透過位移效果，造成不規則狀。另外可以做黑圈圈(顏色加深)來增強視覺效果。

最後用 :before 加上 z-index: 1; 來遮蔽可能凸出的圓形下方。
