window.matchMedia()

可以用來判斷螢幕寬度的方法。

matchMedia() 會回傳一個新的 MediaQueryList 物件，表示指定的媒體查詢字串(CSS @media 規則)解析後的結果。

MediaQueryList 物件以下屬性：

media：查詢敘述的內容。
matches：用於檢測查詢結果，如果檔案匹配 media query 串列，值為 true，否則為 false。

MediaQueryList 可透過監聽，在查詢結果發生變化時，呼叫函式。
